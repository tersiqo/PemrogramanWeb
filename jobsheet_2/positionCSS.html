position: static;

Position static digunakan untuk mengatur element menjadi static secara default. Elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left dan right.:

Elemen ini diatur menggunakan posisi: static;




posistion: relative;

Sebuah element HTML yang menggunakan position relative akan terletak pada posisi normal. Mengatur properti atas, kanan, bawad dan kiri dari elemen yang diposisikan dengan "relatif" akan membuatnya jauh dari posisi normalnya. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.

Elemen ini diatur menggunakan posisi: relative;



position: fixed;

Sebuah element HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Berlaku pengaturan left, bottom, top dan right pada position fixed:

Elemen ini diatur menggunakan posisi: fixed;















position: absolute;

Element HTML yang menggunakan postion absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal.:

Elemen ini diatur menggunakan posisi: relative;
Elemen ini diatur menggunakan posisi: absolute;






position: sticky;

Cobalah untuk scroll untuk mengetahui bagaimana posisi sticky bekerja.
